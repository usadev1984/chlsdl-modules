#+TITLE: chlsdl-modules
#+AUTHOR: usadev1984
#+LANGUAGE: en

modules repo for [[https://github.com/usadev1984/chlsdl][chlsdl]] which hosts modules for all of the supported sources.
** supported sources
- danbooru (images-only)
** dependencies
the full dependencies (some module may not depend on all of these) are:
- pcre2
- json-c
- pkg-config (build-time only)
** building
if you don't want colored output or your terminal doesn't have truecolor support, don't pass ~COLOR=1~ to ~make~.
- build modules:
#+BEGIN_SRC shell
make -j$((`nproc`+1)) modules COLOR=1
#+END_SRC
- install with:
#+BEGIN_SRC shell
  sudo make install
#+END_SRC
*NOTES*:
- to build only two modules where ~$mod_1~ and ~$mod_2~ are the names of the module directories, pass ~MODULES="lib${mod_1} lib${mod_2}"~ to both commands. example:
#+BEGIN_SRC shell
make -j$((`nproc`+1)) modules MODULES="lib${mod_1} lib${mod_2}" COLOR=1
sudo make install MODULES="lib${mod_1} lib${mod_2}"
#+END_SRC
- if you alter the value of ~$PREFIX~, you *MUST* use the same value when building the main program
